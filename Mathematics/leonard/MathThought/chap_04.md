# Newton-Leibniz 之前的欧洲数学

1.  Lenonardo (Fibonacci) (AD1170~1250): 1202年写成 《Liber Abaci》, 对改变中世纪的欧洲数学面貌有很大影响. 汇总了阿拉伯数学(600-1492), 包含阿拉伯数字, Fibonacci 级数
    > Fibonacci 级数的通项表达式存在无理数. 数学中经验主义的失败(研究一种数学现象可能需要一个更大数学结构的帮助)
2.  一般的一元三次方程的 **对称表达式** 的解法:
    1.  给出方程 $x^3 + p x^2 + r x +s = 0$
    2.  进行平移: $x = t - p/3$, 方程成为 $t^3 + bt + c =0$
    3.  令 $t=u+v$ (最简单的二元对称表达式). 从而将方程变换为
        $$
        u^3 + v^3 + c + (3uv + b)(u+v) = 0
        $$
        其解包含了方程组
        $$
        \begin{aligned}
        u^3 + v^3 + c & = 0 \\
        3uv + b &= 0
        \end{aligned}
        $$
        的解. 将 $v = -b/3u$ (第二个方程的解)带入第一个方程, 并换元 $y= u^3$, 给出
        $$
        y + (-b/3)^3 \frac 1 y + c = 0
        $$
        求解此二次方程 (对称表达式中总能保证$y$不为0) 给出 $y$ 的两个根(分别对应 $u, v$), 开三次方给出其他根.(三次根在复平面有三个分支)
    
3.  一般一元四次方程的解法:
    1.  平移消去三次项给出方程 $x^4 + b x^2 + c x + d = 0$
    2.  如果 $c=0$, 则求解关于 $x^2$ 的二次方程
    3.  如果 $c\neq 0$, 则因式分解为 $(x^2+px +k)(x^2-px+j)= 0$, 其中 $y=k+j$ 是某个三次方程的根. 求解此三次方程,然后求解二次方程即得.
